== Description ==

Enhance your WooCommerce store with the Bulk Discount for WooCommerce plugin. This powerful extension allows you to effortlessly implement bulk discount pricing strategies, encouraging customers to buy more and save more. Set up tiered pricing, volume-based discounts, or special promotions for specific products. With easy configuration options and a seamless integration into the WooCommerce platform, this plugin is the perfect solution for driving sales and maximizing customer satisfaction.

== Key Features ==

Set percentage or fixed amount discounts based on the quantity purchased.
Apply  globally or to specific products.

== Installation ==

1. Upload the `bulk-woo-discount` directory to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Configure the plugin settings under WooCommerce > Woocommerce Products discount.

== Frequently Asked Questions ==

= Does this plugin work with the free version of WooCommerce? =

Yes, Bulk Woo Discount is designed to work seamlessly with the free version of WooCommerce.

= How do I configure bulk discounts for my products? =

After activating the plugin, navigate to WooCommerce > Woocommerce Products discount in your WordPress admin.
Now, head to the "All Products" page in WooCommerce.
Select the products to which you want to apply the bulk discount. You can do this by using the checkboxes next to each product
At the top of the product list, locate the "Bulk Actions" dropdown.
Choose "Apply Product discount" from the dropdown menu.
After selecting "Apply Product discount," click the "Apply" button to activate the bulk discount for the selected products.
That's it! You've successfully configured and applied bulk discounts to your WooCommerce products using the Bulk Woo Discount plugin. Monitor your store to ensure that the discounts are applied as expected during the checkout process.
